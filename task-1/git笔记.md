git init

两种链接方式

git remote add origin XXXXXX 

or

git clone XXXXX



git branch 查看本地所有分支
git status 查看当前状态 

git add 添加到工作区

git add -A 将所有变动添加到工作区

git commit  -m""提交 

git checkout dev 切换到本地dev分支
git remote show 查看远程库



git push origin （master） 将本地项目给提交到服务器中

git pull 本地与服务器端同步

git push (远程仓库名) (分支名) 将本地分支推送到服务器上去。
